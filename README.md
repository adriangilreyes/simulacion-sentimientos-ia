Simulación de Sentimiento en una IA
Con la libreria de Diseó Gráfico TKINTER, hemos preparado una pequeña ventana visual, donde podemos encontrar un input, donde el usuario / cliente le da indicanciones a la IA de cómo se siente en ese momento, finalmente en la parte inferior de la vista encontraremos un text-area donde la IA, nos muestra el sentimiento que intridujo el usuario.
